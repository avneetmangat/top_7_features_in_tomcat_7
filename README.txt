This folder contains the tomcat7demo web app to accompany the devx/internet.com article on new features in Tomcat 7.

The contents of this folder are:
src - contains java source code
lib - servlet-api.jar
web - contains jsp and xml file for the web app
build.xml - ant build file
.classpath and .project - Eclipse project files

To build a deployable war file, type 'ant' in command line.  The 'war' file will be available in 'build' folder.
